# Components Index

This document provides a complete inventory of all available components, patterns, and examples in the Bootstrap Agentic Component Library.

## Component Structure

Each component is located in `/components/<component-name>/` and includes:
- `<component-name>.html` - Ready-to-copy HTML snippet
- `<component-name>.md` - Documentation (purpose, variants, usage rules, accessibility)

## Usage Notes

- All components are designed to work with Bootstrap 5.3.8 via CDN
- Components follow Bootstrap's utility-first approach
- Each component includes accessibility best practices
- Components can be combined and customized using Bootstrap utility classes
- Always refer to individual component documentation for specific usage rules and variants
- Patterns provide multi-component recipes for common UX patterns
- Examples demonstrate complete page implementations using components

## Getting Started

1. **Start with App Shell**: Use `/assets/app-shell.html` as base template
2. **Reference Components**: Consult this index to find components
3. **Follow Patterns**: Use patterns for common UX patterns
4. **Study Examples**: Review examples for complete page implementations
5. **Read Documentation**: Always read component `.md` files for usage rules
