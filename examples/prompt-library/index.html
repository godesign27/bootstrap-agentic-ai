Prompt Library Demo

This page demonstrates the Prompt Library drawer component. The drawer provides quick access to all prompts in the repository, organized by category.

How to Use

Keyboard Shortcut: Press Cmd+L (Mac) or Ctrl+L (Windows/Linux) to open the Prompt Library drawer.

Or click the button below:

Features

  • Browse prompts organized by category
  • View full prompt content in a readable format
  • Copy prompts to clipboard with one click
  • Keyboard accessible (ESC to close, Tab to navigate)
  • Focus trap when drawer is open

Adding to Your Project

To add the Prompt Library drawer to any HTML page, include these two lines:

<link rel="stylesheet" href="/assets/prompt-drawer/prompt-drawer.css">
<script type="module" src="/assets/prompt-drawer/prompt-drawer.js"></script>

Note: This must be served via a local web server (not file:// protocol) due to browser security restrictions for fetching JSON and markdown files.
